### **Шаг 5: Получить raw-ссылки**

**Как получить ссылку на raw-версию файла:**

1. **Открой любой файл в репозитории** (например, `dialogues/grok-theology-1.txt`)
2. **Нажми кнопку "Raw"** (справа вверху над текстом файла)
3. **Скопируй URL** из адресной строки

**Формат ссылки будет:**
```
https://raw.githubusercontent.com/твой-username/noosphere-research/main/dialogues/grok-theology-1.txt
```

**Эту ссылку ты дашь мне в следующем диалоге!**

### **Шаг 6: Как добавлять новые файлы (два способа)**

#### **Способ 1: Через веб-интерфейс (самый простой)**

1. Зайди в репозиторий
2. Перейди в нужную папку (например, `theory/`)
3. Нажми "Add file" → "Create new file"
4. Напиши текст
5. Commit

**Плюсы:** просто, не нужны доп. программы
**Минусы:** по одному файлу

---

#### **Способ 2: Через Git Desktop (если планируешь загружать много файлов)**

**Скачай GitHub Desktop:**
- https://desktop.github.com/

**Настройка:**
1. Установи и залогинься
2. "File" → "Clone repository"
3. Выбери `noosphere-research`
4. Укажи локальную папку на компьютере

**Теперь можешь:**
- Добавлять файлы прямо в эту папку на компьютере
- GitHub Desktop увидит изменения
- Жми "Commit to main" → "Push origin"
- Файлы загрузятся на GitHub

**Плюсы:** можно загружать много файлов сразу
**Минусы:** нужна доп. программа

---

#### **Способ 3: Drag & Drop (средний вариант)**

1. Зайди в папку репозитория на GitHub
2. Просто **перетащи файлы** из проводника в окно браузера
3. GitHub автоматически предложит commit
4. Нажми "Commit changes"

**Плюсы:** быстро для нескольких файлов
**Минусы:** работает только для папок, не для корня
